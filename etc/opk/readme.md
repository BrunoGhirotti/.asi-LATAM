# OpenKore
É recomendável baixar o [GordoKore](https://github.com/Brunnexo/gordokore) ou o OpenKore dessa [fonte oficial](https://github.com/openkore/openkore).
Se baixar da fonte oficial, será necessário aplicar o patch presente nesta pasta: copie e cole na pasta raiz do OpenKore, substituindo os arquivos e pastas.
**O GordoKore já tem esse patch!**

## Atenção!
É recomendável fazer o patch antes de qualquer configuração, já que o patch **reescreverá o `config.txt`!**
